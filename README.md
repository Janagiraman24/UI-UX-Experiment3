## Aim:
To perform heuristic evaluation of an existing website and compare its user experience with competing platforms and analyze usability issues and improve the design based on UX principles.
To redesign the interface of the boAt website for better usability.
## Algorithm:
1.Select the boAt website and two competitor websites.

2.Study their UI layout, navigation, and design elements.

3.Apply heuristic principles (visibility, consistency, feedback, etc.).

4.Identify usability issues and compare strengths and weaknesses.

5.Propose improvements for better user experience.

6.Recreate the improved design using HTML and CSS.
